### Projects

  Here are some of the projects I’ve worked on:

- **[WiniWalk Android - Step Counter App](https://github.com/abdelillahbel/WiniWalkAndroid)** : A fitness app that tracks your daily steps and encourages a healthy lifestyle.  
- **[VendVite Express Android - Vendor Management](https://github.com/abdelillahbel/VendViteExpressAndroid)** : A mobile app for managing vendors and streamlining business operations.  
- **[SkillSnap Android - Portfolio Builder](https://github.com/abdelillahbel/SkillSnapAndroid)** : An Android app for creating and managing professional portfolios with SEO-ready pages.  
- **[SkillSnap Web - Web Portfolio Platform](https://github.com/abdelillahbel/SkillSnapWeb)** : A web-based version of SkillSnap for building and sharing portfolios online.  
- **[Flight Booking App - Travel Planner](https://github.com/abdelillahbel/FlightBookingApp)** : A flight booking system with features like seat selection and itinerary management.  
- **[InstaSocial - Social Media App](https://github.com/abdelillahbel/Instasocial)** : A social media platform for sharing photos, videos, and connecting with friends.  
- **[PetFinder ASP Core MVC - Pet Adoption](https://github.com/abdelillahbel/PetFinderASPCoreMVC)** : A web application for finding and adopting pets, built with ASP.NET Core MVC.  
- **[PetFinder MAUI - Cross-Platform Pet Finder](https://github.com/abdelillahbel/PetFinderMAUI)** : A cross-platform app for finding pets, built with .NET MAUI.  
- **[Algeria Towns API - Location Data](https://github.com/abdelillahbel/algeria-towns-api)** : An API providing data about towns and cities in Algeria.  
- **[Storytelling App - Interactive Stories](https://github.com/abdelillahbel/StorytellingApp)** : An app for creating and sharing interactive stories with multimedia elements.  
- **[Image Encryption & Compression - Security Tool](https://github.com/abdelillahbel/ImageEncryptionCompression)** : A tool for encrypting and compressing images to ensure data security.
